
# Terraformers

Terraform and Terraforming Docker Image

# Install

```
# Setting ~/.aws/credentials on the host  
./install.sh [<default-region>]
```

# Usage

```
cd tf
vi *.tf
../terraform [--version] [--help] <command> [<args>]
../terraforming <command> [<args>]
cd ..
```

# Shell

```
./shell.sh
```

# Uninstall

```
./uninstall.sh
```

# Build

```
./build.sh
```
