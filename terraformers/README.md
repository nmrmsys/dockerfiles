
# terraformers

terraform and terraforming Docker Image

# Install

```
./install.sh
```

# Setup

```
./setup.sh
# vi ~/.aws/credentials
```

# Usage

```
cd tf
vi *.tf
../terraform [--version] [--help] <command> [<args>]
../terraforming <command> [<args>]
cd ..
```

# Uninstall

```
./uninstall.sh
```

# Build

```
./build.sh
```
